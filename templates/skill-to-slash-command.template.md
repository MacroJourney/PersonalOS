# 技能晋升模板：知识技能 → Slash Command

> 将 `Atlas/Dots/Skills/` 中的知识技能转化为 `.claude/skills/` 的可执行 slash command。

## 使用方法

1. 确认目标技能满足晋升条件（见 `docs/00_核心设计.md` 复利机制章节）
2. 复制下方模板，填入对应内容
3. 保存到 `.claude/skills/[skill-name]/SKILL.md`
4. 在原知识笔记中添加标注：`> 已晋升为 slash command：/[skill-name]`

## SKILL.md 模板

---

# /[skill-name] — [一句话描述]

[技能的核心目标，1-2 句话说明这个 slash command 做什么]

## 参数

- `$1`：[第一个参数说明]。如未提供，[默认行为]

## 步骤

1. [从知识技能的「步骤」转化：每步写明具体动作]
2. [确保步骤可执行，不是抽象描述]
3. [关键决策点标注等待用户确认]

## 检查清单

- [ ] [从知识技能的质量标准提取]
- [ ] [输出格式符合预期]

## 注意事项

- [从知识技能的「适用场景」和「使用记录」中提炼注意点]
- [标注不适用的场景]

---

## 转化对照

| 知识技能字段 | SKILL.md 对应 |
|------------|--------------|
| 标题 + 描述 | `# /skill-name — 描述` |
| 步骤 | `## 步骤`（转为可执行动作） |
| 适用场景 | `## 注意事项`（含不适用场景） |
| 使用记录 | 不迁移（保留在原知识笔记） |
