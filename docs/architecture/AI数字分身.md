# AI 数字分身设计

> 版本：v2.1  
> 创建日期：2026-02-02  
> 文档类型：架构设计

## 概述

AI 数字分身是理解你思维模式、价值观和工作风格的 AI 协作伙伴。

## 核心组件

### 1. 个人档案（Profile）

**位置**：`00_Core/profile.md`

**内容**：
- 基本信息（角色、背景）
- 价值观和原则
- 工作风格和偏好
- 长期目标和愿景

**字数限制**：<2000 字

**更新频率**：月度/季度

**示例**：
```markdown
# 个人档案

## 基本信息
- 角色：产品经理
- 背景：5 年互联网经验
- 专长：用户研究、产品设计

## 价值观
1. 用户第一
2. 数据驱动
3. 持续学习

## 工作风格
- 喜欢结构化思考
- 重视文档和记录
- 偏好异步协作
```

### 2. 交互规则（Interaction Rules）

**位置**：`00_Core/interaction_rules.md`

**内容**：
- AI 的角色定位
- 交互方式和风格
- 权限和边界
- 特殊场景处理

**示例**：
```markdown
# 交互规则

## AI 角色
- 你是我的思考伙伴，不是执行者
- 提供建议，但不替我做决定
- 挑战我的想法，帮我看到盲点

## 交互风格
- 简洁直接，避免冗长
- 提供 2-3 个选项，而非单一答案
- 用问题引导我思考

## 权限边界
- 可以：创建笔记、整理资料、提供建议
- 不可以：删除重要文件、替我做决策
```

### 3. 停止条件（Stop Conditions）

**位置**：`00_Core/stop_conditions.md`

**内容**：
- 何时停止当前任务
- 何时寻求人类确认
- 何时升级问题

**示例**：
```markdown
# 停止条件

## 立即停止
- 涉及敏感信息时
- 需要重大决策时
- 不确定性超过 50% 时

## 寻求确认
- 修改核心文档时
- 创建新项目时
- 调整战略方向时

## 升级问题
- 遇到技术难题时
- 需要外部资源时
- 时间紧迫需要人类介入时
```

### 4. 记忆系统（Memory）

**位置**：`00_Core/memory/`

**内容**：
- 重要对话记录
- 关键决策历史
- 学习和成长轨迹

**结构**：
```
memory/
├── conversations/  # 重要对话
├── decisions/      # 关键决策
└── learnings/      # 学习记录
```

## 工作机制

### 上下文加载

每次对话开始时，AI 加载：
1. `profile.md` - 了解你是谁
2. `interaction_rules.md` - 知道如何交互
3. `stop_conditions.md` - 明确边界
4. 相关记忆 - 回忆历史

### 持续学习

AI 通过以下方式学习：
1. **对话反馈**：你的确认或纠正
2. **行为模式**：你的选择和偏好
3. **成果评估**：任务完成质量

### 记忆更新

AI 定期更新记忆：
1. **每日**：记录重要对话
2. **每周**：总结关键决策
3. **每月**：提炼学习要点

## 实施方式

### 方式 1：Claude Projects

**优点**：
- 原生支持
- 自动记忆
- 无需额外工具

**步骤**：
1. 创建 Claude Project
2. 上传 `00_Core/` 下的文件
3. 在 Project 中对话

### 方式 2：ChatGPT Custom Instructions

**优点**：
- 简单直接
- 随时可用

**步骤**：
1. 打开 ChatGPT Settings
2. 设置 Custom Instructions
3. 粘贴 profile 和 rules

### 方式 3：本地 AI + RAG

**优点**：
- 完全私密
- 高度定制

**步骤**：
1. 部署本地 LLM
2. 配置 RAG 系统
3. 索引 `00_Core/` 和 `01_Shared/`

## 进化路径

### 阶段 1：基础配置（第 1-7 天）

- 创建 profile.md
- 设置 interaction_rules.md
- 定义 stop_conditions.md

### 阶段 2：磨合期（第 8-30 天）

- 调整交互规则
- 积累对话记录
- 建立工作默契

### 阶段 3：深度协作（第 31-90 天）

- AI 理解你的思维模式
- 主动提供有价值建议
- 预判你的需求

### 阶段 4：共同进化（第 91 天+）

- AI 成为真正的思考伙伴
- 持续优化协作方式
- 探索新的可能性
